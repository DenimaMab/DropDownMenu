Partie CSS

Partie HTML

Pour ajouter un autre niveau il suffit de rajouter une liste après la balise a qui se trouve à l'intérieur li.
L'ordre est très important sinon le script JS sera mal interprété.

Architecture :

  • ul premier niveau
    • li
      • a lien non terminal
        • ul deuxième niveau
        • a lien terminal
